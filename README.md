# LUXE

**LUXE** — A luxury fashion brand website. Timeless. Uncompromising. Since 1892.

Crafted with the elegance of Louis Vuitton, the sophistication of Dior, and the edge of Saint Laurent.

## Design Philosophy
- Pure black background
- Soft ivory typography
- Gold accents (#D6B56C)
- Slow, luxurious animations and transitions
- Cinematic, editorial layouts

## Features
- **Fullscreen cinematic runway video** hero with "Crafted Beyond Trends." headline
- **Mouse-follow spotlight** effect for premium tactile feel
- **New Collection** with 4 signature collections in elegant cards
- **Lookbook** — Magazine-style grid with cinematic image parallax and full-screen modals
- **Atelier / Craftsmanship** — Detailed 4-step timeline of the making process
- **Designer Notes** — Editorial quotes from Creative Director Margot Laurent
- **Journal** — 3 in-depth magazine-style articles
- **Contact** — Elegant private inquiry form
- **Special Interactions**: Slow luxury transitions, typography reveals, parallax scrolling, cinematic modals

## Tech Stack
- React 19
- Tailwind CSS 3
- Framer Motion (for all animations, modals, parallax)
- Vite
- Lucide Icons

## Navigation
- Collections
- Atelier
- Journal
- Campaigns (Lookbook)
